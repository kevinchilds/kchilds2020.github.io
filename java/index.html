What is Java?

Java is an object oriented language based in C and maintained by Oracle

JDK vs JRE vs JVM

Java is a compiled language as opposed to an interpreted language. Essentially what this means is that the code is converted to a more machine readable language called bytecode before the jvm can run it.

What is the JDK?

JDK stands for Java Development Kit. The JDK has everything needed for java development. It allows us to compile our source code to bytecode as well as generate source documentation like javadocs. The JDK is able to do this because the jdk holds a program called a compiler. A compilers purpose is to convert high level language into a lower level language.

What is the JRE?

JRE stands for Java Runtime Environment. The JRE has the necessary libraries that needed for your specific project.

What is a JVM?

JVM stands for Java Virtual Machine. It runs this bytecode line by line
